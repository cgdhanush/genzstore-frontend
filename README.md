# ⚡ GenzStore Frontend

A sleek, modern frontend for **GenzStore**, built with **React.js**. Designed for speed, scalability, and an engaging user experience tailored for the next-gen audience.

---

## 🌐 Overview

The GenzStore Frontend delivers a dynamic and responsive UI for browsing products, managing accounts, and placing orders. It communicates seamlessly with the Spring Boot backend via REST APIs.

---

## 🛠️ Tech Stack

* **React.js (Vite / CRA)**
* **JavaScript (ES6+) / TypeScript (optional)**
* **Redux Toolkit / Context API**
* **Axios / Fetch API**
* **Material UI**
* **React Router**
* **JWT Authentication**

---


## ✨ Features

* 🎯 Clean & responsive UI
* 🔐 Secure Authentication (JWT-based)

## 🔐 Authentication Flow

* User logs in → Backend returns JWT
* Token stored in localStorage / cookies
* Token attached to API requests via Axios interceptors
